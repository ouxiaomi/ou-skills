# ou-skills

> Skills for vibe coding, such as Claude Code or Codex

这是我（ouxiaomi）的个人技能仓库，按分类组织管理的 AI 技能集合。

## 📁 目录结构

```
ou-skills/
├── categories/              # 分类目录
│   ├── code-analysis/      # 代码分析类技能
│   ├── development/        # 开发工具类技能
│   ├── productivity/       # 生产力工具
│   └── communication/      # 通信相关
├── SKILLS-INDEX.md         # 技能索引（查看详情）
└── README.md              # 本文件
```

## 🚀 快速开始

### 查看所有技能

查看 [SKILLS-INDEX.md](./SKILLS-INDEX.md) 获取完整的技能列表和分类信息。

### 使用技能

每个技能目录包含：
- **SKILL.md** - 技能说明和使用指南（必需）
- **reference.md** - 参考文档（可选）
- **examples.md** - 使用示例（可选）
- **scripts/** - 辅助脚本（可选）
- **templates/** - 模板文件（可选）

### 创建新技能

1. 参考现有技能的结构
2. 查看 `categories/development/template-skill/` 中的模板
3. 遵循标准技能模板格式
